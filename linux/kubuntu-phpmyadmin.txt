# Setup phpMyAdmin di Kubuntu Desktop (Nginx + MariaDB + PHP)

## Install package

```bash
sudo apt update
sudo apt install nginx mariadb-server php php-fpm php-mysql php-mbstring php-xml php-gd
```


# Install Bersih MariaDB 11.4 di Ubuntu 24.04

## Install dependency

```bash
sudo apt install curl gnupg
```

---

## Tambahkan key MariaDB

```bash
sudo mkdir -p /etc/apt/keyrings

curl -fsSL https://mariadb.org/mariadb_release_signing_key.pgp \
| sudo gpg --dearmor -o /etc/apt/keyrings/mariadb-keyring.pgp
```

---

## Tambahkan repo MariaDB 11.4

```bash
sudo nano /etc/apt/sources.list.d/mariadb.list
```

Isi:

```bash
deb [signed-by=/etc/apt/keyrings/mariadb-keyring.pgp] https://mirror.mariadb.org/repo/11.4/ubuntu noble main
```

---

## Update repo

```bash
sudo apt update
```

---

## Install MariaDB 11.4

```bash
sudo apt install mariadb-server mariadb-client
```

---

## Jalankan service

```bash
sudo systemctl enable mariadb && sudo systemctl restart mariadb
```

---

## Cek versi

```bash
mariadb -V
```

Harus muncul:

```text
MariaDB 11.4
```

---



# Ubah Password Root MariaDB untuk phpMyAdmin

Masuk MariaDB:

```bash
sudo mysql
```

Ubah password root:

```sql
ALTER USER 'root'@'localhost' IDENTIFIED VIA mysql_native_password USING PASSWORD('phpmyadmin');
FLUSH PRIVILEGES;
EXIT;
```

Restart MariaDB:

```bash
sudo systemctl restart mariadb
```



---

## Buat symlink phpMyAdmin

```bash
sudo ln -s /usr/share/phpmyadmin /var/www/phpmyadmin
```

---

## Buat virtual host Nginx

```bash
sudo nano /etc/nginx/sites-available/phpmyadmin.local
```

Isi:

```nginx
server {
    listen 80;
    server_name phpmyadmin.local;

    root /var/www/phpmyadmin;
    index index.php index.html;

    location / {
        try_files $uri $uri/ =404;
    }

    location ~ \.php$ {
        include snippets/fastcgi-php.conf;
        fastcgi_pass unix:/run/php/php8.3-fpm.sock;
    }
}
```

---

## Aktifkan site

```bash
sudo ln -s /etc/nginx/sites-available/phpmyadmin.local /etc/nginx/sites-enabled/
```

---

## Tambahkan hosts lokal

```bash
sudo nano /etc/hosts
```

Tambahkan:

```text
127.0.0.1 phpmyadmin.local
```

---

## Restart service

```bash
sudo systemctl restart nginx
sudo systemctl restart php8.3-fpm
```

---

## Login

Buka:

```text
http://phpmyadmin.local
```

Login phpMyAdmin:

```text
user: root
password: phpmyadmin
```


---

## Hilangkan error controluser phpMyAdmin

```bash
sudo nano /etc/phpmyadmin/config.inc.php
```

Comment bagian ini:

```php
// $cfg['Servers'][$i]['controluser'] = 'phpmyadmin';
// $cfg['Servers'][$i]['controlpass'] = 'password';
```

Tambahkan bagian ini di bawah komen tadi:

```
$cfg['Servers'][$i]['AllowNoPassword'] = false;
$cfg['Servers'][$i]['hide_db'] = 'mysql|information_schema|performance_schema|test|phpmyadmin|sys';
```

Tambahkan bagian ini sebelum "End of servers configuration":

```
$cfg['DefaultCharset'] = 'utf8';
$cfg['DefaultConnectionCollation'] = 'utf8mb4_general_ci';
$cfg['MaxNavigationItems'] = 1000;
$cfg['MaxTableList'] = 1000;
$cfg['NavigationTreeEnableGrouping'] = false;
$cfg['NavigationDisplayLogo'] = false;
```

Simpan.

Restart:

```bash
sudo systemctl restart nginx
sudo systemctl restart php8.3-fpm
```
